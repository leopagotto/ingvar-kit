## üéØ Usage Examples

### Import Direct @ingka Components

```typescript
// Option 1: Import from ingka-direct.ts
import { 
  Button, 
  Card, 
  InputField,
  Switch,
  RadioButton 
} from '@ingvar-kit/skapa-components/ingka-direct';

// Use with official Skapa APIs
<Button variant="primary" size="medium">Click me</Button>
<InputField label="Email" type="email" />
<Switch checked={isOn} onChange={setIsOn} />
```

### Import from Combined Index (Future)

```typescript
// Option 2: Import from index-new.ts (dual exports)
import { 
  Button as IngkaButton,     // Official @ingka
  Button,                    // Our wrapper
  InputField,                // Official @ingka
  TextField                  // Our wrapper
} from '@ingvar-kit/skapa-components/index-new';

// Choose which version per use case
<IngkaButton variant="primary">Official</IngkaButton>
<Button variant="primary">Wrapper</Button>
```

### Import Custom Wrappers (Current)

```typescript
// Option 3: Import from index.ts (current default)
import { 
  Button,      // Our wrapper
  TextField,   // Our wrapper (wraps InputField)
  Toggle,      // Our wrapper (wraps Switch)
  Radio        // Our wrapper (wraps RadioButton)
} from '@ingvar-kit/skapa-components';

// Simplified APIs
<Button variant="primary" size="large">Click me</Button>
<TextField label="Email" value={email} onChange={setEmail} />
```
